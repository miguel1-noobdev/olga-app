# Datos que queremos obtener para el inventario botánico

Este archivo resume qué información nos interesa conseguir de fuentes externas.

## Campos objetivo

| Campo | Descripción | Ejemplo |
|-------|-------------|---------|
| Nombre común | Nombre cotidiano de la planta | Lavanda |
| Nombre científico | Nombre taxonómico | Lavandula angustifolia |
| Familia botánica | Clasificación taxonómica | Lamiaceae |
| Parte usada | ¿Qué parte de la planta se aprovecha? | Flores, hojas, raíz |
| Compuestos químicos naturales | Principios activos principales | Linalool, acetato de linalilo, cineol |
| Propiedades | Efectos beneficiosos | Relajante, antiséptico, cicatrizante |
| Contraindicaciones | Advertencias de uso | Primer trimestre de embarazo, alergia al linalool |
| Tipos de extracto | Formas de extracción casera | Macerado, hidrolato, aceite esencial |

## Notas

- Hay que revisar la licencia de cada fuente antes de scrapear.
- Dr. Duke's (USDA) permite descarga directa en CSV/Excel, es la opción más segura.
- Las fuentes médicas suelen estar orientadas a uso interno; nosotros las adaptamos a uso cosmético externo.

## Reglas tentativas de extracción casera

- Si la planta destaca por aceites esenciales / volátiles → hidrolato o aceite esencial en gotas.
- Si destaca por carotenoides o vitaminas liposolubles → macerado en aceite (oleomacerado).
- Si destaca por mucílagos o taninos → macerado en agua / infusión / mucílago.
