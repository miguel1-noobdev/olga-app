# Visión general del proyecto

## ¿Para quién es?

Para mi amiga, que hace cosmética natural artesanal en casa.

## ¿Qué problema resuelve?

Hoy no tiene un lugar centralizado donde guardar:
- sus fórmulas,
- los procesos que sigue,
- las observaciones de cada lote.

Eso le impide mejorar recetas, repetir éxitos y no volver a cometer errores.

## ¿Qué es?

Una aplicación privada tipo laboratorio digital.

## ¿Qué NO es?

- No es un marketplace.
- No es para vender online.

## Funcionalidades que podría tener (sin orden definitivo)

1. Inventario botánico: plantas, compuestos, propiedades, contraindicaciones, extractos.
2. Calculadora de fórmulas: ingredientes con porcentajes, adaptación al tamaño del lote.
3. Bitácora de lotes: fecha, fórmula base, observaciones, fotos, mediciones.
4. Web pública informativa: resultados, fotos, beneficios, divulgación sobre cosmética natural.
5. Scraper o importador de datos botánicos desde fuentes confiables.

## Dudas abiertas

- ¿De dónde sacamos datos confiables de plantas?
- ¿Scraping, carga manual, o dataset abierto?
- ¿Qué stack técnico usar?
- ¿Primero la app privada, la web pública, o el inventario botánico?

## Fuentes para investigar

- Dr. Duke's Phytochemical and Ethnobotanical Databases (USDA) - permite descarga CSV.
- Fitoterapia.net.
- Atlas Virtual de Plantas Medicinales (UMH).
- Compendium of Botanicals (EFSA).
