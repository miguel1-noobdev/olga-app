# Ideas - Olga App

Este directorio es un espacio de trabajo para la fase de tormenta de ideas.
Nada de lo que está aquí es definitivo. Se guarda todo, se depura después.

## Estructura

```
ideas/
├── raw/         # Notas, conversaciones, recortes, ruido de IA. No se edita, se acumula.
├── resumen/     # Ideas ya destiladas por nosotros. Acá va lo que vale la pena recordar.
└── descartado/  # Cosas que no van, pero que no queremos borrar todavía.
```

## Reglas

- `raw/`: pegá acá lo que venga de chats, webs, libros, screenshots. No hace falta ordenar.
- `resumen/`: escribí acá con tu voz. Un párrafo corto alcanza. El formato es Markdown.
- `descartado/`: si algo de `raw/` o `resumen/` ya no va, mové el archivo acá en lugar de borrarlo.
